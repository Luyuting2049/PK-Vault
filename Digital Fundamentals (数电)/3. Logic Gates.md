# The Invertor (非门)
非运算：高低电平互换
多个非门放一起：反码

# The AND Gate
可以大于2个输入：所有都是HIGH才是1
表示：X = A·B or X= AB（和乘法比较像，所以说到A乘B就是说AB与运算）
**用法**：mask：保留想要的bit位（比如八位想保留最后四位，和00001111相乘）
画waveform把都HIGH的部分画成1，剩下全是

# The OR Gate
也可以多个输入，均为LOW时输出为0
X = A+B（后面读起来方便可能会直接这么说）
**用法**：把某个bit位上的强制变成1

# The HAND Gate (与非门)
先做与运算再做非运算，但是独立的基本逻辑门
（圆圈代表非运算，所以它符号是与门后面加个圈）
waveform：都HIGH的找出来输出LOW，剩下都HIGH
**特点**：通用逻辑门（用这一个逻辑门可以实现其他逻辑门功能）
	非：输入
