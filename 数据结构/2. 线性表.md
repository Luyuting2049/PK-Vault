线性结构（又称线性表）
	数据成员具有相同特征
	每个元素都有唯一前趋/后继
相关术语：
	N：表的大小（具体成员个数）
	$A_0$：首结点，$A_N-1$ 尾节点

属性类：常量成员函数

扩大容量的实现：
（开更大容量、把原来的放到新数组）
![[Pasted image 20260306092836.png|306]]
```cpp
template <class elemType>
void seqList<elemType>::doubleSpace()
{
	elemType *tmp = data; //tmp指针指向data所指空间
	maxSize *= 2; 
	data = new elemType[maxSize]; 
	for (int i = 0; i < currentLength; ++i) 
		data[i] = tmp[i]; 
	delete [] tmp;//勿忘
}
```
data（指针变量）：栈
new开出的空间：堆
